# Six Shoot Word Game

一个西部风格的英文词语/句子猜字母游戏。玩家需要在枪手愤怒值达到 100% 或 6 次错误之前猜出答案。

## 玩法

- 选择题型：`词语` 或 `句子`
- 点击屏幕键盘或使用实体键盘输入字母
- 猜对会显示对应字母
- 猜错会装填一发子弹
- 6 次错误会失败，并显示“哦不！”
- 每一关完成后会显示“恭喜！”，然后自动进入下一 Level
- 枪手愤怒值会在 1 分 30 秒内从 0% 增长到 100%
- 愤怒值达到 100% 时会自动失败

## 功能

- 关卡自动推进
- 词语与句子两种题型
- 每关不同类别与提示
- 西部枪手场景
- 愤怒值计时系统
- 胜利、失败、上膛音效
- 无需安装依赖，可直接打开运行

## 本地运行

直接打开 `index.html`，或者运行本地服务器：

```bash
python3 -m http.server 5174
```

然后访问：

```text
http://127.0.0.1:5174/
```
